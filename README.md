# My Simple Website

A clean, modern, and responsive website template built with HTML, CSS, and JavaScript.

## Features

- Responsive design that works on all devices
- Modern and clean user interface
- Smooth scrolling navigation
- Contact form with basic validation
- Mobile-friendly navigation
- Easy to customize and extend

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Start customizing the content to make it your own!

## Customization

### HTML Structure
- Modify the content in `index.html` to add your own text and sections
- Add or remove navigation items in the `<nav>` section
- Customize the contact form fields as needed

### Styling
- Edit `styles.css` to change colors, fonts, and layout
- The CSS is organized by sections for easy modification
- Color variables can be found at the top of the file

### JavaScript
- The `script.js` file contains basic functionality for:
  - Smooth scrolling
  - Contact form handling
  - Navigation highlighting
- Add your own JavaScript functionality as needed

## Browser Support

This template works in all modern browsers including:
- Chrome
- Firefox
- Safari
- Edge
